・本作品はゲームパッド操作を推奨しています。
・XBOXコントローラーでの動作を確認しています。

//***********************************************
// キーボード & マウス 操作
//***********************************************
・画面遷移 				→ [ Enter ]
・移動     				→ [ W/A/S/D ]
・ジャンプ 				→ [ Space ]
・アクション 			→ [ マウスの左クリック ]
・ターゲット切り替え 	→ [ マウスの右クリック ]
・カメラ操作			→ [ マウスのフリック移動 ]
・ポーズ 				→ [ P ]
・ポーズ操作 			→ [ ↑/↓ or W/S ]

//***********************************************
// ゲームパッド 操作
//***********************************************
・画面遷移 				→ [ A / Start ]
・移動     				→ [ Lスティック / 十字キー ]
・ジャンプ 				→ [ A ]
・アクション 			→ [ RT ( 右後ろトリガー ) ]
・ターゲット切り替え 	→ [ LB / RB ]
・カメラ操作			→ [ Rスティック ]
・ポーズ 				→ [ Start ]
・ポーズ操作 			→ [ ↑/↓ ]
